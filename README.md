# Dijital Project

### Welcome to Dijital Project


## Description

* Site created using Webpack, SCSS, GSAP


### Running
1. Clone the repository and enter the 

```bash
$ git clone git@github.com:iraand/dijital-project.git && cd dijital-project 
```


### Getting up and running

To install use:

```
npm i
```

To start use:

```
npm run dev
```
